CSS颜色代码大全

基础说明

当设置文本颜色时，CSS 中的 color 属性可以接受以下几种参数形式：

  • 颜色名称
    • 使用预定义的颜色名称来设置文本颜色，例如 red、blue、green 等。
  • 十六进制值
    • 使用六位十六进制值来设置文本颜色，例如 #FF0000 表示红色。
  • RGB 值
    • 使用 RGB 值来设置文本颜色，例如 rgb(255, 0, 0) 表示红色。
  • RGBA 值
    • 使用 RGBA 值来设置文本颜色和透明度，例如 rgba(255, 0, 0, 0.5) 表示半透明的红色。

需要注意的是，除了以上四种形式外，还可以使用 HSL、HSLA、HEX、RGB 等其他颜色表示形式。不同的表示形式在使用上略有差异，但其本质都是用来定义颜色的。

例如，以下是使用以上四种形式分别设置文本颜色的示例代码：

color: red; /* 使用颜色名称 */
color: #FF0000; /* 使用十六进制值 */
color: rgb(255, 0, 0); /* 使用 RGB 值 */
color: rgba(255, 0, 0, 0.5); /* 使用 RGBA 值，其中最后一个参数表示透明度，取值范围为 0 到 1 */

不同的参数形式可以满足不同的需求，在使用时需要根据实际情况进行选择。
